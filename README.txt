代码仓库（code）使用说明

代码仓库中存放项目进展过程中产生的相关文件，包括经过整理的文本，产生的研究思路，研究过程中的代码文件等

团队成员可根据需要自行建立文件夹存放自己的文件

（注意：为了避免混乱，在存放文件时需要在本文档后添加说明，格式：姓名-时间-文件名、文件夹名）

------------------------------------------------------------------------------------------------

项目（project）使用说明

此页主要针对项目进展进行相关安排

项目成员可根据自己的需要自行建立各自需要的项目进展表格，进行进展规划

（注意：在没有特殊情况的时候，请勿修改项目整体进展表格）
------------------------------------------------------------------------------------------------

问题（issue）使用说明

此页主要针对项目进行过程中产生的需要协调沟通的问题

如果遇到无法解决，需要帮助的问题；或是产生了新的想法需要讨论的

项目成员可根据需要自行在issue版块中提出

------------------------------------------------------------------------------------------------
